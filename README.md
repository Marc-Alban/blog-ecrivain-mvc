# 📖 Blog Écrivain MVC

![PHP](https://img.shields.io/badge/PHP-777BB4?style=for-the-badge&logo=php&logoColor=white)

> "Billet simple pour l'Alaska" - Un blog développé entièrement en PHP avec une architecture MVC classique, incluant un espace administration et un système de commentaires.

## 🚀 Lancement avec Docker
```bash
docker-compose up -d --build
```
Rendez-vous ensuite sur `http://localhost:8080/`.
